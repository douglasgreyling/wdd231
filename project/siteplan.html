Site Name

Movie Tracker

This name helps describe the key purpose for this website.

Site Purpose

The website will be a catalog of information about movies. The website would have a page to view a list of movies. Users can search through the list by filtering the movie title. Movies can be clicked on to read more about the plot, and mark it as watched.

Scenarios

  • What movies can I track?

  • What movies haven't I watched yet?

Colour Schema

Overall the theme of the website will need to look something similar like a clapperboard used in filming movies.

Because of this the following colours will be used:

  • Black (for the page's background color, and the colour used for hovering links)

  • White (for the text color, and the background colour over hovered links)

Typography

The font used for the site will be Roboto. This font will be used for all text on the site.
